# Technical Reference

This section contains technical specifications, environment variable details, deployment models, and metrics references for the `s3lim` analysis engine.

To help you configure and audit your deployments, we provide the following reference guides:

1. **[Configuration Guide]({{< relref "configuration.md" >}})**: Detailed documentation of all environment variables, execution thresholds, and configuration settings for the Lambda function.
2. **[Analysis Triggers]({{< relref "triggers.md" >}})**: Documentation of all invocation triggers supported by the `s3lim` Lambda, including S3 Event Notifications, Scheduled CloudWatch Events (cron), manual payloads, and SQS DLQ redrives.
3. **[Deployment Specs]({{< relref "deployment.md" >}})**: AWS SAM template parameters, outputs, and IAM requirements for the Autopilot and Standard deployment stacks.
4. **[Available Metrics]({{< relref "metrics.md" >}})**: Reference of all CloudWatch custom metrics, dimensions, and namespaces generated by the core analysis engine.

## Quick Reference Summary

Use the guides below depending on your immediate need:

* **Need to change Lambda behaviors or thresholds?** Check out the [Configuration Guide]({{< relref "configuration.md" >}}).

* **Auditing IAM roles or cloud formation parameters?** View the [Deployment Specs]({{< relref "deployment.md" >}}).
* **Building a custom CloudWatch dashboard?** Browse the [Available Metrics]({{< relref "metrics.md" >}}).
